this website

it all began at around the end of my 8th grade year. i had passion for web development before then but never actually got to work and made a website. mostly due to me not having a proper domain, i just didn't have the money for it. then i found infinityfree. i don't reccomend it unless you are really desperate. it gave me a .rf.gd domain, which i was fine with. the website took off and it was way bare bones. i didn't know much about html, css and i did not know any javascript. it was poorly coded, with inline this after that and so forth. but that did not stop me from having a website and having fun!

i knew one of my mom's co-workers. he was super chill and really wanted me to learn and learn and learn! he told me about the github student developer pack. it's awesome and i totally reccomend checking it out! once 9th grade hit i actually got a real domain (the one your on right now) and my website was open source for the first time. here i learned more and more and more css and javascript. next iternation over the other and so much time spent.

i used to host my website on github pages and it was okay. i had github on my phone when i used to host it and i got constant notifications that my builds had failed and i had no clue why. i'd reccomend github pages if you just want everything in one spot. currently though, i use cloudflare pages. i love it so much! it works so well and i don't really have any major complaints about it.
